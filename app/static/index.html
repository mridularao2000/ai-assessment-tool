AI Assessment System

Developer control panel

Workflow: Create Curriculum (→ assessment auto-generated & emailed) → Get Assessment with ID + token from email → Submit Answer → Get Results. For reschedule, use the same assessment ID + token.
1Create Curriculum

At least one of notes, links, GitHub repos, or a file is required.

2Get Assessment

The assessment ID and token arrive in your scheduled email. During local testing you can find the token in the assessment table (submission_token column).

3Submit Answer
4Get Results

Grading runs asynchronously after submission. If you get a 404, wait a moment and retry.

5Reschedule Assessment

Only active or scheduled assessments can be rescheduled. The reason is classified by Claude — medical, interview, and emergency requests are approved; procrastination and lack-of-preparation are denied.
